## 2026-03-15

### 주요 버그 수정 및 개선

- **qmd 검색 결과 경로 매칭 개선**
    - qmd가 반환하는 경로(소문자+하이픈)와 실제 파일 경로(대문자+언더스코어) 불일치로 인해 Vector/Deep 검색 결과에서 파일을 열 수 없는 문제를 fuzzy 매칭으로 해결
    - fuzzy 매칭: 소문자 통일, `_`와 `-`를 동일하게 취급하여 볼트 내 실제 파일을 찾음
- **Deep 검색 오류 안내 개선**
    - Deep 검색(전체 컬렉션) 시 리랭킹 컨텍스트 초과로 SIGABRT 발생 시, 스택 트레이스 대신 사용자 친화적 한국어 안내 메시지로 표시
    - Deep 검색에서 전체 컬렉션 선택 시 사전 경고 Toast 추가
- **에러 메시지 UI 개선**
    - 에러 메시지 줄바꿈을 실제 단락으로 표시하여 가독성 향상
- **검색 결과 콘솔 로그 추가**
    - Deep/Vector/BM25 각 검색 결과의 file, collection, relativePath, title을 콘솔에 출력하여 경로 포맷 차이 디버깅 가능
